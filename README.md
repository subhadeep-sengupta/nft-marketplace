# NFT Marketplace

Here you can List and Buy MPL Core NFTs trustlessly, powered by Solana Blockchain and Escrow implementation


## Quick Start

### 1. Clone the Repo
```bash
git clone https://github.com/subhadeep-sengupta/nft-marketplace.git
cd nft-marketplace
```

### 2. Build and Deploy the Program
Make sure you have Anchor Installed and Configured

```bash
anchor build 
anchor deploy
```
### 3. Run the tests
Install dependencies and run tests using yarn

```bash
yarn install
anchor test
```
This will build the Solana program, deploy it to your local validator or configured cluster, and run the full test suite including minting, listing, buying, and delisting NFTs.
